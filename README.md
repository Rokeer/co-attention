# co-attention
Code for BEA 13 paper "Co-Attention Based Neural Network for Source-Dependent Essay Scoring"

## Dependencies

python 2 for data/preprocess_asap.py

python 3 for the rest

* keras 2.2.4
* tensorflow 1.12.0
